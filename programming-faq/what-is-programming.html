ما هي البرمجة ؟

البرمجة ( Programming ) هي عملية كتابة مجموعة من الأوامر و الأكواد للكمبيوتر ( أو أي جهاز إلكتروني أخر مثل السيارات و الهواتف المحمولة ) لكي يقوم بتنفيذ وظيفة معينة .. و يقوم المبرمج بتحديد هذه الوظيفة مسبقا .

طريقة كتابة الأكواد للكمبيوتر و مكان كتابتها :

طريقة الكتابة :

يقوم المبرمجين بكتابة الأكواد البرمجية عن طريق لغات مخصصة يستطيع الكمبيوتر فهمها و التعامل معها و هذه اللغات تسمي لغات البرمجة ( Programming Languages ) .. و التي يترجمها الكمبيوتر إلي لغته الأساسية و هي لغة الألة ( Binary Language ) أو كما يطلق عليها البعض ( لغة الصفر و الواحد ) .. و هذا يعني أن أي كود يكتبه المبرمج يقوم الكمبيوتر بترجمته لأمر من الاثنين إما صفر أو واحد .. و يرمز الصفر إلي عدم وجود تيار كهربي .. بينما يرمز الواحد إلي وجود تيار كهربي .. و هذه أمثلة لأشهر لغات البرمجة في الفترة الحالية : Python , PHP , JavaScript و كل لغة يمكن استخدامها في مجال معين من مجالات البرمجة .

مكان كتابة الأكواد :

يمكننا كتابة الأكواد البرمجية في برامج مخصصة تسمي محررات أكواد ( Code Editors ) و تكون عبارة عن بيئة تطوير متكاملة ( IDE ) و هي اختصار ل ( Integrated Development Environment ) .. و تحتوي هذه البرامج علي إضافات ( Plugins ) لتسهيل و تسريع عملية كتابة الأكواد علي المبرمج و يمكننا تخصيص اختصارات في الكيبورد ( Shortcuts ) بداخل هذه البرامج لتقوم بعمل وظيفة معينة بمجرد الضغط علي حرف أو أحرف معينة في الكيبورد مما يساعد المبرمج علي إنجاز عمله بشكل أسرع .. و من أشهر محررات الأكواد : Visual Studio Code , Brackets , Atom .. و برنامج VS Code التابع لشركة مايكروسوفت هو الأكثر استخداما .

كيف تتعلم البرمجة في أسرع وقت ؟

إذا كنت تريد تعلم البرمجة فأمامك ثلاث طرق مختلفة و يمكنك اختيار ما يناسبك علي حسب مميزات و عيوب كل طريق :

الطريق الأول :

هو دراسة البرمجة بشكل أكاديمي و لك بأن تتعلم أساسيات البرمجة كاملة باستخدام لغة C++ و هي من أقدم اللغات البرمجية و أكثرهم تعمقا في مفاهيم البرمجة الأساسية .. و هذا هو السبب الذي يجعل معظم الكليات الخاصة بعلوم الحاسب تبدأ بدراسة لغة C++ في السنة الأولي للطلاب داخل الكلية .. و أيضا C++ هي اللغة المستخدمة في إنشاء نظام التشغيل Windows التابع لشركة Microsoft و تكون هي اللغة الأساسية المستخدمة في إنشاء أي برنامج يعمل علي نظام Windows .. و إذا قررت أن تبدأ في تعلم البرمجة من خلال هذا الطريق فسوف تعمل بعد فترة كمهندس برمجيات Software Engineer .. و سوف تكون قادر علي تعلم مجالات البرمجة المختلفة بكل سهولة لأنك تعلمت الأساسيات الكاملة من خلال لغة C++ .

الطريق الثاني :

هو أنك تبدأ بتعلم لغة برمجة بسيطة و سهلة من حيث المفاهيم الخاصة بها و طريقة كتابتها مثل لغة بايثون Python .. و Python هي المركز الأول من حيث رغبة التعلم تبعا لبعض الإحصائيات في المواقع المشهورة .
